# Bridgelet FAQ

**What is Bridgelet?**
Bridgelet provides ephemeral Stellar accounts that let organizations pay recipients who do not yet have wallets. Funds sweep into a permanent wallet when the recipient claims.

**Is Bridgelet production ready?**
Bridgelet is in early development (MVP phase). Expect breaking changes and evolving APIs.

**Where do I report a security issue?**
Please follow `SECURITY.md` and email aminubabafatima8@gmail.com. Do not open public issues for vulnerabilities.

**Where should I ask usage questions?**
Use GitHub Discussions for how-to questions, integration guidance, or design feedback.

**Where do I file bugs?**
Open a GitHub Issue with steps to reproduce, expected vs. actual behavior, logs, and environment details.

**Do you offer commercial support?**
Not at this time. For partnership conversations, reach out via email.
